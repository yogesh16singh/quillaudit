## Installation.

- Clone the repository.

```bash
    git clone https://github.com/Zeddxx/quillaudits.git
```

- Get into the repository.
```bash
   cd quillaudits
```

- Install the package.

```bash
    npm install or yarn install or pnpm install
```

- Run the project.

```bash
    npm run dev or yarn dev
```
